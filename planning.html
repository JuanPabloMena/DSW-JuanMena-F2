Contenido

Basicamdente CSS3 es el lenguaje con el cual aplicamos la capa de presentacion a nuestro sitio web, hagamos un ejemplo con un carro, html seria la estructura del carro pero CSS seria la representacion de como queremos que el cliente o la otra persona observe el carro, como el tamaño de las ventanas, color, detalles, etc. Permite aplicar caracteristicas de presentacion que antes eran muy dificiles de lograr sin ayuda de las imagenes

Las hojas o estilos

Existen 3 tipos de hojas o estilos:

  • Hoja de estilo externa: Esta almacena en un archivo diferente al del archivo con el código HTML al cal estar vinculado a través del elemento link, que debe ir situado en la sección head. Es la manera de programar mas eficiente, ya que separa completamente las reglas de formato para la página HTML de la estructura básica de la página.
  • Hoja de estilo interna: est incorporada a un documento HTML, a través del elemento style dentro de la sección head, consiguiendo de esta manera separar la información del estilo del código HTML
  • Hoja de estilo en linea: El Estilo en Línea sirve para insertar el lenguaje de estilo directamente dentro de la sección body con el elemento style. Sin embargo, este tipo de estilo no se recomienda pues se debe intentar siempre separar el contenido de la presentación.

Selectores CSS

Especifica que elementos HTML van a estar afectados por esa declaración, de manera que hace de enlace entre la estructura del documento y la regla estilística en la hoja de estilo. En pocas palabras estos definen sobre qué elementos se aplicará un conjunto de reglas CSS. Los selectores mas basicos son:

  • Selector de tipo: hace coincidir los elementos por nombre de nodo. En otras palabras, selecciona todos los elementos del tipo dado dentro de un documento.
  • Selector de clase: hace coincidir los elementos en función del contenido de su atributo. (class)
  • Selector de ID: coincide con un elemento en función del valor del (id) atributo del elemento . Para que el elemento sea seleccionado, su (id) atributo debe coincidir exactamente con el valor dado en el selector.
  • Selector universal: (*) coincide con elementos de cualquier tipo.
  • Selector de atributos: hace coincidir elementos en función de la presencia o el valor de un atributo determinado.
